# khalidalnujaidi.github.io

Personal site of **Khalid Alnujaidi** — Computer Scientist & Systems Architect.

A hyper-minimalist "Engineering Passport": raw semantic HTML5 + a single vanilla
CSS file. **Zero JavaScript frameworks. No web fonts, no trackers, no analytics,
no cookie banners.** System font stacks only; designed to load near-instantly.

## Structure

```
index.html                       Landing page (passport + essay stream)
main.css                         The one stylesheet (system fonts, light/dark)
feed.xml                         RSS 2.0 feed
404.html                         Minimal not-found page
robots.txt
essays/
  tenure-bottleneck.html         Essay
concepts/
  stalking-the-wild-pendulum/    Interactive essay (self-contained)
projects/
  refund-audit.html              Multimodal Refund Audit System (Delivery Hero)
style.css                        Legacy stylesheet used by projects/refund-audit.html
```

## Deploy

Served by GitHub Pages from the `master` branch root. Push to `master` and it goes live.
