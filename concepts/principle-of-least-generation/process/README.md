# /process — the making-of

Working notes, experiment plans, and reproduction scaffolding behind
**The Principle of Least Generation** series. The published chapters live one
level up (`../chp1.html`, `../chp2.html`); this folder is where each chapter is
designed, pre-registered, and logged before its findings are written into the page.

- [`CHAPTER2_LEASHED_DECODER.md`](CHAPTER2_LEASHED_DECODER.md) — pre-registered
  experiment plan for Chapter 02 (Rung 3: the grammar-leashed slot extractor).
  Written **before** the runs; findings get transcribed back into `../chp2.html`.

Chapter 01's reproduction code lives in its own `cbyte/` (gate + coverage +
baselines); Chapter 02 adds `cbyte/ch02/` on top of it (see the plan).
